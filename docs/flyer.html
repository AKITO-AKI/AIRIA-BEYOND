困ったときは

プレリリースでよくある“つまずき”の回避策をまとめました。

  1. ログインが「未設定」と出る → OAuthのClient ID未投入（運用側の設定）
  2. 生成が止まった → 画面の「生成を再開」で復帰できます
  3. 時間がない → オンボーディングは「創作から」だと短い
  4. 規約/プライバシー → 下のリンクから確認できます
